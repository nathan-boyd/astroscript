# astroscript

### install app

`go install pandamonk.org/astroscript`

### run astroscript

`astroscript removeJpgs`

### run tests

`go test ./...`

### run app without installing

`go run main.go`

### add new console command

`cobra add remove-jpgs --config .cobra.yaml`

### app generated with

`cobra init --pkg-name pandamonk.org/astroscript`
